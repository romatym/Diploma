Описание системы поддержки пользователей FAQ (дипломный проект PHP-24 Нетологии Работа в системе поддержки реализована в двух режимах: Пользователь - может задавать вопросы и смотреть ответы Администратор - может редактировать все данные вопросы, ответы, категории и список администраторов На начальной странице показаны категории вопросов слева и список вопросов с разбиением по категориям по центру. У вопроса есть заголовок и текст вопроса. Заголовки вопросов видны сразу, текст вопроса виден при клике на заголовок. Для чтения ответов на вопрос, нужно перейти по ссылке “Смотреть ответы”. Чтобы работать в режиме Администратора, нужно авторизоваться, перейдя по ссылке “Войти” в левом верхнем углу экрана. Логин: admin Пароль: admin После авторизации администратор получает доступ к ссылке “Вход в админзону”, и возможность редактировать, скрывать, удалять вопросы прямо на главной странице. В админзоне можно редактировать администраторов, категории вопросов, и работать со списком вопросов, на которые не были даны ответы. Веб-версия: https://docs.google.com/document/d/1BeqDp0gGB2ubr_i3ngSTLpZ0vBA-NA-Z2OAEJguQzBA/edit?usp=sharing Дамп базы данных находится в корневом каталоге проекта: faq.sql
